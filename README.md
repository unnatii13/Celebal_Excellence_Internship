# Basic Data Exploration and Cleaning using Pandas

## Objective
Perform data exploration and cleaning on an e-commerce dataset using Pandas.

## Tasks Performed
- Loaded CSV dataset
- Explored data using head(), info(), describe()
- Handled missing values
- Removed duplicates
- Filtered rows
- Selected columns
- Created derived column total_amount
- Saved cleaned dataset
